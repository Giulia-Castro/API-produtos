##  API de Produtos

API REST desenvolvida em C# com ASP.NET Core para gerenciamento de produtos, com operações CRUD completas.

###  Tecnologias
- C#
- ASP.NET Core
- Swagger / OpenAPI
- .NET 8

###  Como rodar
1. Clone o repositório
2. Execute:
   dotnet run
3. Acesse a documentação interativa:
   http://localhost:5167/swagger

###  Funcionalidades
- Criar produto (POST)
- Listar todos os produtos (GET)
- Buscar produto por ID (GET)
- Atualizar produto (PUT)
- Deletar produto (DELETE)
- Documentação automática via Swagger
